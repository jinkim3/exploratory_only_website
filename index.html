Exploratory Only is an intuitive tool for conducting large-scale exploratory analyses.

It helps researchers quickly and easily conduct an exhaustive set of
correlation, moderation, and mediation analyses among all variables of interest.


The manuscript introducing Exploratory Only is currently under review at Behavior Research Methods.

Try using Exploratory Only with the Web Application or by downloading the standalone program.
